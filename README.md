## 👤 About Me

IT professional building a public portfolio of hands-on labs that prove the skills behind the certifications. Each lab simulates a real-world IT support scenario — complete with help desk tickets, troubleshooting challenges, and documented resolutions using the STAR method.

I couldn't find labs that matched what I wanted for my portfolio — so I built my own. I'm documenting everything publicly so other aspiring IT professionals have a place to start.
